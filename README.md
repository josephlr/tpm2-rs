# Rust TSS for TPM2

The main `tpm` crate is the TSS (and OS-specific bindings to the TPM).

The `simulator` crate contains an abstraction around Microsoft's
[TPM2 simulator](https://sourceforge.net/projects/ibmswtpm2/).
